# [MongoDB](https://docs.mongodb.com/manual/) with [Spring](https://spring.io)

## The concept in Mongo

| 名词 | 含义 |  
| --- | --- |  
| database | 数据库 |  
| collection | 数据库表/集合 |  
| document | 数据记录行/文档 |  
| field | 数据记字段/域 |  
| index | 索引 |  
| primary key | 主键，自动将_id作为主键 |
